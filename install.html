AlertLight Web Installer

Flash AlertLight firmware directly to your ESP32-S3 device
No software installation required - just use your Chrome browser!

Before You Start

  • Use Chrome, Edge, or Opera browser (Web Serial API required)
  • Have an ESP32-S3 device (QFN56, 8MB PSRAM, 16MB Flash)
  • Connect device via USB-C cable
  • Close Arduino IDE or other serial monitor programs

What Will Be Installed

  • AlertLight v1.0.0 firmware
  • Complete monitoring system for Ukrainian air alerts
  • Power outage schedule tracking (Yasno API)
  • RGB LED notifications with customizable colors
  • Web configuration interface on port 8080
  • 1.47" display with three-screen LVGL UI

  • Configure your WiFi credentials and preferences
  • Device will connect to your network and start monitoring
